      • saveData

        public void saveData(byte[] data,
                     java.lang.String... path)
              throws java.io.IOException
        Записывает данные из массива байтов в файл указанный во втором аргументе. Данные будут перезаписывать данные
        Parameters:
        data - массив данных, которые нужно записать.
        path - путь до файла в который нужно записать данные. Путь может как существовать, так и нет. Если в конце пути добавить расширение, то будет создан файл с этим расширением. Например если передать массив строк ["data","file",".txt"], то данные будет записанный в файл по пути "/pathToResources/data/file.txt". Если передавать массив ["data","file.txt"] или одну строку "/data/file.txt", данные также будут записанный в файл по пути "/pathToResources/data/file.txt"
        Throws:
        java.io.IOException
